Running instructions for the game

1. Install python 3 or better
2. Use "pip install numpy" on the command prompt
3. Use "pip install json" on the command prompt
4. Use "pip install pillow" on the command prompt
5. Use "pip install tkinter" on the command prompt]
6. Navigate in the command prompt to this folder using the "cd" command on windows
7. Use "py project.py" on the command prompt or run project.py through some other application which can run python applcations such as IDLE which comes with your installation of python
